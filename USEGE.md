# Руководство по запуску и использованию SurfSense

SurfSense — это мощная AI-платформа с открытым исходным кодом, предназначенная для работы с вашими документами и данными. Она решает многие ограничения Google NotebookLM, предоставляя полный контроль над данными, отсутствие лимитов, отказ от привязки к одному вендору и поддержку более 25 внешних источников.

## 🚀 Варианты запуска

Существует три основных способа начать работу с SurfSense: облачная версия, самостоятельный хостинг (Self-Hosted) на своем оборудовании и использование настольного приложения.

### 1. Облачная версия (Самый быстрый способ)
Для использования облачной версии не требуется установка:
1. Перейдите на сайт [surfsense.com](https://www.surfsense.com) и авторизуйтесь.
2. Перейдите к коннекторам и подключите необходимые источники (Google Drive, OneDrive, Notion, Slack и т.д.).
3. Включите периодическую синхронизацию, чтобы поддерживать актуальность данных.
4. Пока данные из коннекторов индексируются, вы можете вручную загрузить дополнительные документы.
5. После завершения индексации вы можете начать работу, задавать вопросы и использовать ИИ на основе ваших данных.

### 2. Самостоятельный хостинг (Self-Hosted)
Разверните SurfSense на собственных серверах для полного контроля над данными и обеспечения максимальной приватности.

**Требования:** У вас должен быть установлен и запущен [Docker Desktop](https://www.docker.com/products/docker-desktop/) (или Docker Engine).

**Установка для Linux / MacOS:**
```bash
curl -fsSL https://raw.githubusercontent.com/MODSetter/SurfSense/main/docker/scripts/install.sh | bash
```

**Установка для Windows:**
```powershell
irm https://raw.githubusercontent.com/MODSetter/SurfSense/main/docker/scripts/install.ps1 | iex
```
*Примечание: Установочный скрипт автоматически настраивает Watchtower для ежедневных автоматических обновлений. Чтобы пропустить этот шаг, добавьте флаг `--no-watchtower`.*

**После установки:**
Скрипт автоматически скачает файлы в папку `./surfsense` и запустит контейнеры в фоновом режиме. После завершения загрузки платформа будет доступна по следующим адресам:
- **Веб-интерфейс (Frontend):** [http://localhost:3929](http://localhost:3929)
- **API (Backend):** [http://localhost:8929](http://localhost:8929)
- **Документация API:** [http://localhost:8929/docs](http://localhost:8929/docs)

**Управление контейнерами:**
- **Настройка:** Отредактируйте файл `./surfsense/.env` для добавления API-ключей и других параметров.
- **Просмотр логов:** `cd ./surfsense && docker compose logs -f`
- **Остановка:** `cd ./surfsense && docker compose down`

Для более сложной настройки (через Docker Compose вручную) изучите [официальную документацию](https://www.surfsense.com/docs/).

### 3. Настольное приложение (Desktop App)
Настольное приложение позволяет использовать возможности SurfSense глобально, в любой программе на вашем компьютере.
Скачайте актуальную версию со страницы [последних релизов на GitHub](https://github.com/MODSetter/SurfSense/releases/latest).

**Возможности приложения:**
- **General Assist (Общий помощник):** Мгновенный вызов SurfSense из любого приложения с помощью глобального сочетания клавиш.
- **Quick Assist (Быстрый помощник):** Выделите текст в любом окне, затем попросите ИИ объяснить его, переписать или выполнить любое другое действие.
- **Extreme Assist:** Получайте встроенные подсказки от ИИ прямо при наборе текста в любом приложении. Ваши данные используются как контекст для генерации.
- **Watch Local Folder (Отслеживание папок):** Синхронизируйте локальные папки с вашей базой знаний (отлично подходит для папок Obsidian).

## 💡 Основные возможности и сценарии использования (Use Cases)

После того как ваши данные проиндексированы, вам открываются следующие возможности:

1. **Поиск и цитирование (Search and Citation)** - Задавайте вопросы по загруженным данным и получайте точные ответы с кликабельными ссылками на первоисточники.
2. **Вопросы по конкретным документам (Document Mention QNA)** - Отмечайте определенные документы в чате, чтобы ИИ анализировал только их.
3. **Генерация отчетов** - Создавайте детализированные отчеты и экспортируйте их в форматы: PDF, DOCX, HTML, LaTeX, EPUB, ODT, Plain Text.
4. **Создание подкастов** - Генерируйте аудио-дискуссии (подкасты) с участием виртуальных ведущих на основе ваших материалов.
5. **Генерация изображений и видео** - Используйте встроенные агенты для создания медиаконтента по контексту ваших заметок.

## 🤝 Совместная работа в реальном времени (Beta)
Вы можете использовать SurfSense для командной работы:
1. Перейдите на страницу **Manage Members** и создайте приглашение для коллег.
2. Когда участник присоединится по ссылке, выбранное рабочее пространство (SearchSpace) станет доступным для него.
3. Включите общий доступ к нужным чатам (Make chat shared).
4. Теперь вы можете общаться с ИИ и коллегами в реальном времени.
5. Оставляйте комментарии к ответам ИИ или документам и упоминайте (@тегайте) членов команды.
